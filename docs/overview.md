# Factile Documentation Overview

This repository owns the local-first Factile CLI, its user guidance, and its
self-contained implementation tests.

The CLI reads root-local content, mounted local directories, and read-only Git
repositories materialized in a generated per-root cache. Hosted source
resolution and cloud services are outside this repository.
