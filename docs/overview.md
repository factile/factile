# Factile Documentation Overview

This repository owns the local-first Factile CLI and the open contracts that
define portable OKF content, roots and sources, reader behavior, and local
writer behavior.

Hosted product contracts remain in the private `factile-platform` repository
until their external surfaces are stable enough to publish.
